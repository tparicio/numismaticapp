# API Documentation

The Numismatic App exposes a RESTful API for all operations.

## OpenAPI Specification

For the complete, interactive documentation of endpoints, parameters, and schemas, please refer to the OpenAPI 3.0 specification file:

[openapi.yaml](../openapi/openapi.yaml)

## Interaction

The API is consumed by the Vue.js frontend but can also be accessed via any HTTP client (Postman, cURL).

### Base URL
`http://localhost:8080/api/v1`

### Authentication
*Currently, the API is open and does not require authentication for local development.*
